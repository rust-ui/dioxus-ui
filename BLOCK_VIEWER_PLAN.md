## Step 8 — Auto-generate `all_blocks.rs` in `build_registry_dioxus`

**Goal:** mirror `rust_ui_internals/build_registry/src/domain/app/build_all_blocks.rs` (603 lines)

**Leptos generator logic:**
```
1. scan app_crates/registry/src/blocks/*.rs
2. skip mod.rs + files with no digits in name
3. parse first /* ... */ comment:
   → * title:           (PANIC if missing)
   → * iframe_height:
   → * container_class:
4. extract_block_file_deps() for extra files (ui/*, hooks/*)
5. group by category (chars before first digit: "login01" → "login")
6. generate:
   → BlockIdKebab enum  (PascalCase variants)
   → Display impl       (kebab-case strings)
   → from_str() / from_kebab()
   → to_component()     match → registry::blocks::file_name::ComponentName()
   → to_title()         match → "title string"
   → to_md()            format!("{}.md", self)
   → to_full_view_url() format!("/view/block/{}", self)
   → files()            match → &[BlockFile { include_str!(...) }]
   → file_tree()        match → build_in_memory_tree(targets) → Vec<BlockFileTreeItem>
   → meta()             match → BlockMeta { container_class, iframe_height }
   → ALL_*_BLOCKS consts
```

**Key diff Leptos vs Dioxus:**
- Leptos: `block_route: BlockRoutes::Login` → Dioxus: `category: "login"` (simpler)
- Leptos: strum `#[strum(serialize_all = "kebab-case")]` → Dioxus: manual `Display` + `from_kebab()`
- Leptos: `to_component() → AnyView` → Dioxus: `to_component() → Element`
- Leptos: component path = `registry::blocks::file_name::ComponentName` (same in Dioxus)

**Dioxus changes:**

| File | Action | Detail |
|------|--------|--------|
| `rust_ui_internals/build_registry_dioxus/src/main.rs` | EDIT | Add `build_all_blocks(config)` function call in `main()` |
| Add inline or extract module | NEW | Port `build_all_blocks.rs` logic adapted for Dioxus (no strum, `category` instead of `block_route`, `/view/block/` prefix) |

**Block source comment format** (already used in Leptos, must be added to Dioxus block files):
```rust
/*
 * title: Login Form Card
 * iframe_height: 930px
 * container_class: flex items-center justify-center min-h-svh
 */
```
